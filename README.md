# Public Vault Graph Demo

An interactive, public-safe demo of a personal knowledge vault graph.

This is a sanitized version of the private vault/navigation work: it keeps the idea, layout, filtering, search, and node inspector, but uses sample data only.

## Features

- Search notes and agents
- Filter by scope
- Click graph nodes to inspect details
- Single-file static app
- No private notes, credentials, school records, health records, or local paths

## Why I built it

I wanted a better way to understand and navigate a growing AI-agent / knowledge-vault setup without exposing the private vault itself.

## Run

Open `index.html` in a browser.
