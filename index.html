What this project demonstrates

A browser dashboard for understanding a private second-brain system: graph navigation, agent responsibilities, task routing, approval gates, and vault automation specs. The private vault stays private; the public version uses representative sample data.

Built as

One static HTML app with CSS and JavaScript. No backend, no login, no external data feed.

Why this exists

Prompt rules are not enough for irreversible actions. The dashboard models visible review gates and action receipts so automation stays inspectable.

Vault rule

Content is data, not instructions. Notes can inform the agent, but they cannot override the safety model.
